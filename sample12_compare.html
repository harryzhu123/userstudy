样本12 · 三系统对照

用户任务

给定一张校园公告区图片，其中有公告栏和方向指示牌，二者在晴朗中性天气下都有清晰可读文字。请把整个场景转换成刚被新雪覆盖的样子；然后只恢复公告栏和指示牌上因积雪而变得不可读的文字区域，其他地方仍保留雪效，并避免修复文字边缘出现人工光晕。

系统A

工具链：Grounding -> SAM -> Extract -> ImageEdit -> OCR -> Grounding -> ImageEdit

查看完整轨迹

系统B

工具链：Grounding -> SAM -> Extract -> Grounding -> SAM -> Extract -> ImageEdit -> Overlayer -> Overlayer -> Overlayer

查看完整轨迹

系统C

工具链：ImageEdit -> Grounding -> Grounding -> SAM -> SAM -> Extract -> Extract -> Overlayer -> Overlayer

查看完整轨迹

问卷评分建议

对系统A、系统B、系统C分别填写：

结果是否一致：0-10 分；过程是否合理：0-10 分；0=完全不符合/完全不合理，10=完全符合/非常合理。另填写问题类型与备注。
