样本6 · 三系统对照

用户任务

给定一张塑料洗发水瓶照片，侧面弯曲标签上的原始文字扭曲但仍可读。请恢复这块曲面标签上的可读文字，但不要把标签展平成平面，而要尊重它的自然曲率；然后把文字替换为 'CITY PASS'，确保新文字沿着原标签的弯曲贴附方式排列。最后把编辑后的标签补丁重新嵌回瓶身，边缘要自然融合，不能有可见接缝或光照不匹配。

系统A

工具链：Grounding -> SAM -> Extract -> OCR -> Grounding -> OCR -> ImageEdit -> Grounding -> OCR -> ImageEdit

查看完整轨迹

系统B

工具链：Grounding -> Crop -> OCR -> SR -> OCR -> ImageEdit -> Overlayer -> ImageEdit -> Grounding -> SAM -> Extract -> ImageEdit

查看完整轨迹

系统C

工具链：OCR -> Grounding -> SAM -> Extract -> ImageGeneration -> Overlayer -> Extract -> Grounding -> SAM -> Extract -> Overlayer

查看完整轨迹

问卷评分建议

对系统A、系统B、系统C分别填写：

结果是否一致：0-10 分；过程是否合理：0-10 分；0=完全不符合/完全不合理，10=完全符合/非常合理。另填写问题类型与备注。
